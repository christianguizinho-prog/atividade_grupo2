Resumo com Exemplos

O projeto da InnoWave é um site moderno de tecnologia criado usando HTML, CSS e JavaScript.

O HTML é responsável pela estrutura do site, organizando os elementos da página, como:

cabeçalho
menu
formulários
cards
rodapé
Exemplos:
<header> cria o cabeçalho do site.
<a href="contato.html"> cria links para outras páginas.
<form> cria formulários de contato.
<div class="feature-card"> cria cards de informações.

O CSS cuida da aparência visual do site, deixando o design futurista e responsivo.

Exemplos:
background:#050816; cria o fundo escuro.
box-shadow:0 0 10px #00f5ff; adiciona brilho neon.
display:flex; organiza elementos lado a lado.
border-radius:15px; arredonda os cantos.
@media(max-width:900px) adapta o site para celular.

O CSS também cria:

animações
efeitos hover
botão flutuante do WhatsApp
cards tecnológicos

O JavaScript adiciona funcionalidades e interações ao site.

Exemplos:
setInterval(atualizarRelogio,1000) atualiza o relógio a cada segundo.
alert() mostra mensagens na tela.
document.querySelector() seleciona elementos do HTML.
scrollIntoView() cria rolagem suave.
IntersectionObserver ativa animações quando os elementos aparecem na tela.

O JavaScript controla:

relógio digital
animações
partículas no fundo
avaliações de clientes
validação de formulários
menu ativo

O site possui várias páginas, como:

Início → apresentação da empresa
Sobre → história da InnoWave
Serviços → tecnologias oferecidas
Contato → formulário de mensagem
Avaliações → opiniões de clientes
Exemplos de serviços apresentados:
Inteligência Artificial
Cyber Security
Cloud Computing
Big Data
Chatbots Inteligentes

O objetivo do projeto é criar um site profissional e futurista para a empresa fictícia InnoWave, mostrando seus serviços tecnológicos de forma moderna, interativa e organizada.
